## MDX란?

MDX는 Markdown 문서 안에서 JSX를 사용할 수 있게 해주는 포맷입니다. 일반적인 Markdown 문법과 함께 React 컴포넌트를 직접 임포트하고 사용할 수 있습니다.

### 주요 특징

- **Markdown + JSX**: 마크다운의 단순함과 React의 강력함을 결합
- **컴포넌트 재사용**: 반복되는 요소를 컴포넌트로 만들어 재사용
- **타입 안정성**: TypeScript와 함께 사용 가능

## 코드 예시

TypeScript 코드 예시입니다:

```tsx
interface BlogPost {
  title: string;
  date: string;
  content: string;
}

const post: BlogPost = {
  title: "Welcome",
  date: "2025-12-10",
  content: "Hello, world!"
};
```

인라인 코드도 사용할 수 있습니다: `const hello = "world"`
