React 19는 여러 혁신적인 기능을 도입했습니다. 가장 주목할 만한 변화들을 살펴보겠습니다.

## use 훅

React 19에서 가장 큰 변화 중 하나는 `use` 훅의 도입입니다.

### useContext 대체

기존 `useContext`를 대체하는 더 강력한 훅입니다:

```tsx
// React 18 (기존)
import { useContext } from 'react';
const value = useContext(MyContext);

// React 19 (새로운 방식)
import { use } from 'react';
const value = use(MyContext);
```

### Promise 지원

`use` 훅은 Promise를 직접 처리할 수 있습니다:

```tsx
import { use } from 'react';

function UserProfile({ userPromise }) {
  const user = use(userPromise);
  return <div>{user.name}</div>;
}
```

## forwardRef 제거

더 이상 `forwardRef`가 필요 없습니다. ref를 일반 props로 전달할 수 있습니다:

```tsx
// React 18 (기존)
import { forwardRef } from 'react';
const Input = forwardRef((props, ref) => {
  return <input ref={ref} {...props} />;
});

// React 19 (새로운 방식)
function Input({ ref, ...props }) {
  return <input ref={ref} {...props} />;
}
```

## Server Actions

서버에서 실행되는 함수를 클라이언트에서 직접 호출할 수 있습니다:

```tsx
'use server';

export async function createPost(formData: FormData) {
  const title = formData.get('title');
  const content = formData.get('content');

  await db.posts.create({
    title,
    content,
  });
}
```

클라이언트에서 사용:

```tsx
'use client';

import { createPost } from './actions';

export default function PostForm() {
  return (
    <form action={createPost}>
      <input name="title" />
      <textarea name="content" />
      <button type="submit">Submit</button>
    </form>
  );
}
```

## useOptimistic

낙관적 업데이트를 쉽게 구현할 수 있는 새로운 훅입니다:

```tsx
import { useOptimistic } from 'react';

function TodoList({ todos }) {
  const [optimisticTodos, addOptimisticTodo] = useOptimistic(
    todos,
    (state, newTodo) => [...state, newTodo]
  );

  async function addTodo(formData) {
    const newTodo = { id: Date.now(), text: formData.get('text') };
    addOptimisticTodo(newTodo);
    await saveTodoToServer(newTodo);
  }

  return (
    <form action={addTodo}>
      {optimisticTodos.map(todo => (
        <div key={todo.id}>{todo.text}</div>
      ))}
      <input name="text" />
      <button type="submit">Add</button>
    </form>
  );
}
```

## 결론

React 19는 개발자 경험을 크게 개선했습니다. 특히 `use` 훅과 Server Actions는 게임 체인저입니다.
